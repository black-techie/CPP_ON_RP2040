1. install cmake
2. download pico sdk from > https://github.com/raspberrypi/pico-sdk.git
    NB : make sure to export pico sdk path > $ export  PICO_SDK_PATH=/home/usr/....

3. create builld dir 
4. inside build dir > run cmake ..
5. Run make

copy and paste the uf2 file > cp blink.uf2 /media/black/RPI-RP2/

There you go!
